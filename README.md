# SENTINEL: Strategic Intelligence Architect

### *Forensic Synthesis & Agentic Litigation Orchestration*

**SENTINEL** is a high-fidelity legal-tech platform engineered for the 2026 legal landscape. It serves as a **Strategic Architect**, bridging the gap between massive local dockets and real-time global intelligence. By utilizing a multi-tiered agentic framework, it automates the forensic synthesis of evidence, identifies strategic litigation patterns, and maintains an immutable audit trail for every insight generated.

---

## 🏛 Core Architecture

The system is built on a "Logic Bridge" philosophy, ensuring that complex legal queries are decomposed into surgical sub-tasks executed by specialized AI agents.

* **Logic Bridge (LawBridge):** A proprietary translation layer that converts raw user intent into high-precision legal queries.
* **Agentic Orchestration:** A conductor-style system that manages specialized sub-agents for document review, web-based intel retrieval, and cross-border linguistic analysis.
* **Forensic Synthesis Engine:** Dynamically correlates data from local PDF Dockets with live 2026 search results to surface contradictions, precedents, and strategic opportunities.
* **Tiered Failover Infrastructure:** High-availability design that hot-swaps between **Google AI Studio (Gemini 3)** and **OpenRouter (Tier 2)** to bypass rate limits during critical trial preparation.

---

## 🛡 Key Features

* **Access Docket (RAG):** Native support for deep-context ingestion of court filings, transcripts, and transactional records.
* **Real-Time Intel Retrieval:** Integrated Tavily & DuckDuckGo search modules focused on "Current Status 2026" verification.
* **Forensic Audit Trail:** Every assistant response is anchored by a unique **SHA-256 Forensic Hash**, ensuring non-repudiation and data integrity.
* **Temporal Sync:** Real-time monitoring of temporal cycles to ensure all strategic insights are synchronized with the latest legal deadlines and status updates.
* **Polyglot Engine:** Advanced multi-language processing for international litigation and multi-jurisdictional discovery.

---

## 🛠 Tech Stack

* **Intelligence:** Gemini 3 Flash / 1.5 Pro (Primary), Meta Llama 3.3 (Failover via OpenRouter).
* **Framework:** Streamlit (UI), LangChain / LangSmith (Traceability), TextBlob (Sentiment & NLP).
* **Vector Infrastructure:** Supabase & HuggingFace Embeddings.
* **Search:** Tavily API / DDGS.

---

## 🚀 Getting Started

1. **Clone the Architecture:**
```bash
git clone https://github.com/SENTINEL-Legal-tech/Sentinel.git

```


2. **Synchronize Environment:**
Ensure your `.env` contains `GEMINI_KEY`, `META_KEY`, and `TAVILY_API_KEY`.
3. **Initiate Sentinel:**
```bash
streamlit run centurion.py

```

> **Forensic Note:** This platform is designed for professional legal environments where precision and auditability are non-negotiable. It implements strict guardrails against "Shadow AI" actions through persistent monitoring and human-in-the-loop checkpoints.
